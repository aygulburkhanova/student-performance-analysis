# Student Performance Analysis 🎓

Проект по исследованию зависимостей оценок студентов от различных факторов (часы обучения, уровень стресса) с использованием машинного обучения.

## Используемые модели:
1. **Линейная регрессия** — поиск прямой зависимости.
2. **Полиномиальная регрессия** — моделирование нелинейных связей.
3. **Нейронная сеть (MLP)** — оптимизация функции потерь.

## Как запустить:
1. Склонируйте репозиторий.
2. Создайте окружение: `python -m venv .venv`.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите основной скрипт.
